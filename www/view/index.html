Ufront

The client-server framework that lets you build the next big web app.

Current state: Ufront has been in development for a while and is moving towards a stable release. Many features are in there, some are still being developed, and quite a few need more thorough documentation.

Client and server

Write in one language on the client and server. Share models, views and controllers between server and client. Even call server APIs from client side code.

Currently supported platforms are Neko, PHP and Browser JS. We'd love to add NodeJS, C++ and the JVM and Python to that list.

A powerful framework

Ufront at it's heart is an MVCA framework - you structure you code in Models, Views, Controllers and APIs. This lets you write well structured code - which is great for collaboration, testing, and for staying sane as your project grows.

We're also make it easy to plug in extra functionality with dependency injection and a middleware framework.

A language you'll love

Ufront is powered by Haxe - a powerful programming language with an emphasis on safety, speed and cross platform portability.

Side note: while Ufront is powered by Haxe, the Haxe website is powered by Ufront!
